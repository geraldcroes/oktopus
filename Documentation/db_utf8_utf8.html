\Oktopus\UTF8

Package: Default

A port of Kohana_UTF8 (http://kohanaframework.org/) wich is a port of [phputf8](http://phputf8.sourceforge.net/) to a unified set of files. Provides multi-byte aware replacement string functions.

For UTF-8 support to work correctly, the following requirements must be met:

  • PCRE needs to be compiled with UTF-8 support (--enable-utf8)
  • Support for Unicode properties is highly recommended (--enable-unicode-properties)
  • UTF-8 conversion will be much more reliable if the iconv extension is loaded
  • The mbstring extension is highly recommended, but must not be overloading string functions

from_unicode( array $arr ) : stringboolean
static

Takes an array of ints representing the Unicode characters and returns a UTF-8 string.

Astral planes are supported i.e. the ints in the input can be > 0xFFFF. Occurrances of the BOM are ignored. Surrogates are not allowed.

$str = UTF8::to_unicode($array);
